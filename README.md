# CyxbsMobile_Android

## 如何编译
- 请先查看必看文档中的多模块教程，熟悉多模块开发规范
- 对 `module_main` 模块进行单模块编译
  - ⚠️注：`module_app` 模块只用于打正式包，因为包含密钥等文件，所以只能由副站持有
